AntiCVP-Deep: Identify anti-coronavirus peptides between different negative datasets based on self-attention and deep learning


###AntiCVP-Deep uses the following dependencies:

Python 3.7
numpy
scipy
scikit-learn
pandas
TensorFlow
keras
